# Security Policy

## Supported versions

Only the latest release on the `master` branch is supported.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report it privately through GitHub's
[private vulnerability reporting](https://github.com/foXaCe/lovelace-card-mod/security/advisories/new).

Target response time: 7 days.

Because card-mod is a frontend plugin that injects CSS and evaluates
user-provided templates in the browser, please include the affected Home
Assistant frontend version and a minimal reproduction when relevant.
